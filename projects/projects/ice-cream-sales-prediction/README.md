## Ice Cream Sales Prediction

This project focuses on predicting ice cream sales revenue based on temperature data using a machine learning regression model.

### Key Highlights
- Regression-based prediction problem
- Data preprocessing and exploratory analysis
- Model training and evaluation
- Model persistence using Pickle
- Streamlit-based deployment for real-time predictions

### Technologies Used
Python, Pandas, NumPy, Scikit-learn, Matplotlib, Streamlit
